## 🚀 Пошаговый процесс запуска

### Этап 1: Подготовка (5 минут)

```
1. Откройте Microsoft Excel
2. Создайте новый файл
3. Сохраните как Roulette_Game.xlsm
4. Откройте VBA редактор (Alt + F11)
```

**Время**: 5 минут  
**Статус**: ⏳ Подготовка

---

### Этап 2: Добавление кода (10 минут)

```
1. Создайте 3 новых модуля в VBA
2. Скопируйте код из src/RouletteGame.bas
3. Скопируйте код из src/GameInterface.bas
4. Скопируйте код из src/SpreadsheetLayout.bas
5. Закройте редактор VBA (Alt + Q)
```

**Время**: 10 минут  
**Статус**: ⏳ Добавление кода

---

### Этап 3: Инициализация (2 минуты)

```
1. Откройте Alt + F8 (Macros)
2. Найдите SetupGame
3. Нажмите Run
4. Появятся листы "Игра" и "История"
```

**Время**: 2 минуты  
**Статус**: ⏳ Инициализация

---

### Этап 4: Первый запуск (3 минуты)

```
1. Откройте лист "Игра"
2. Нажмите "НАЧАТЬ ИГРУ"
3. Введите сумму ставки (1-100)
4. Выберите тип ставки
5. Посмотрите результат
```

**Время**: 3 минуты  
**Статус**: ✅ Готово!

**Общее время**: 20 минут

## ❓ FAQ (Часто задаваемые вопросы)

### В: Насколько безопасна игра?
**О**: Это симуляция для образовательных целей. Деньги виртуальные.

### В: Могу я изменить начальный баланс?
**О**: Да, отредактируйте ячейку B1 перед нажатием "НАЧАТЬ ИГРУ".

### В: Как добавить звуки?
**О**: Положите WheelOfRulete.wav в папку с игрой и добавьте модуль SoundEffects.bas.

### В: Можно ли играть на нескольких компьютерах?
**О**: Да, скопируйте файл Roulette_Game.xlsm на другой компьютер.

### В: Как экспортировать результаты?
**О**: Используйте кнопки "ОЧИСТИТЬ", "ИСТОРИЯ" или код в CODE_EXAMPLES.md.

### В: Какая максимальная ставка?
**О**: По умолчанию 100, можно изменить в коде.

### В: Работает ли на Mac?
**О**: Да, если установлен Excel для Mac (требуется Excel 2016+).

### В: Нужен ли интернет?
**О**: Нет, всё работает локально.
