## Добавление VBA кода

### Шаг 1: Откройте редактор VBA

```
Способ 1 (быстро):
Windows: Alt + F11
macOS: Option + Fn + F11

Способ 2 (через меню):
Developer → Visual Basic
(Если вкладки Developer нет, включите её:
 File → Options → Customize Ribbon → 
 Поставьте галку Developer)
```

**Результат:** Откроется окно VBA редактора (слева дерево проекта, справа - редактор кода).

### Шаг 2: Создайте первый модуль (RouletteGame)

В окне редактора VBA:

```
1. Правый клик на проект (слева в дереве)
2. Insert → Module
3. Создастся новый модуль (например Module1)
4. Переименуйте его (если хотите):
   Правый клик на Module1 → Properties
   (Name): RouletteGame
```

**Результат:** Пустой модуль готов для кода.

### Шаг 3: Скопируйте код RouletteGame.bas

**В текстовом редакторе (Notepad/VS Code):**

```
1. Откройте файл src/RouletteGame.bas
2. Ctrl + A (выделить все)
3. Ctrl + C (скопировать)
```

**В VBA редакторе:**

```
1. Нажмите в основное поле редактора (справа, где пусто)
2. Ctrl + V (вставить весь код)
3. Код появится в модуле
```

**Результат:** Весь код из RouletteGame.bas загружен в первый модуль.

### Шаг 4: Создайте второй модуль (GameInterface)

```
Повторите шаг 2 для второго модуля:
1. Правый клик на проект
2. Insert → Module
3. Переименуйте в GameInterface (опционально)
4. Скопируйте код из src/GameInterface.bas
5. Вставьте в редактор (Ctrl + V)
```

### Шаг 5: Создайте третий модуль (SpreadsheetLayout)

```
Повторите процесс:
1. Правый клик на проект → Insert → Module
2. Переименуйте в SpreadsheetLayout
3. Скопируйте код из src/SpreadsheetLayout.bas
4. Вставьте в редактор (Ctrl + V)
```

**Результат после Шага 5:** 

```
В окне VBA (слева в дереве):
├── Project (Roulette_Game.xlsm)
│   ├── Microsoft Excel Objects
│   │   ├── Sheet1
│   │   └── ThisWorkbook
│   └── Modules
│       ├── RouletteGame           ✅
│       ├── GameInterface          ✅
│       └── SpreadsheetLayout      ✅
```

Все три модуля с кодом загружены!

## Инициализация игры

### Шаг 1: Закройте редактор VBA

```
Способ 1: Alt + Q
Способ 2: Нажмите крестик на окне редактора
```

Вы вернетесь в Excel.

### Шаг 2: Откройте Macros Dialog

```
Windows: Alt + F8
macOS: Option + Fn + F8

Или через меню:
Developer → Macros
(или View → Macros → View Macros)
```

**Откроется окно:**

```
┌───────────────────────────────────┐
│ Macro Name:          │ [SetupGame]  │
│                      │              │
│ Макросы:             │              │
│ - SetupGame          │ [Run]    ← Нажмите здесь
│ - InitializeGame     │ [Cancel] │
│ - PlayRoulette       │ [Edit]   │
│                      │ [Create] │
│ Макросы в:           │ [Delete] │
│ [Roulette_Game.xlsm] │          │
└───────────────────────────────────┘
```

### Шаг 3: Выполните макрос SetupGame

```
1. В списке найдите "SetupGame"
2. Нажмите кнопку [Run] (или двойной клик)
```

**В процессе выполнения:**
- Может появиться окно с сообщением
- Excel слегка "мигнет"
- Создадутся новые листы

**После выполнения:**

```
Появится сообщение:
┌─────────────────────────────┐
│ Игровой макет создан        │
│ успешно!                    │
│                             │
│ [OK]                        │
└─────────────────────────────┘
```

Нажмите **[OK]**.

### Шаг 4: Проверьте результат

В нижней части Excel (где вкладки листов) должны появиться:

```
[Sheet1] [Игра] ← Новый лист    [История] ← Новый лист
```

**Отлично! Игра установлена успешно!** ✅

## Решение проблем

### ❌ Проблема: "Макросы отключены" или "Безопасность макросов"

**Решение:**

```
Способ 1 (быстро):
В информационной панели сверху нажмите [Enable Macros]

Способ 2 (постоянно):
File → Options → Trust Center → Trust Center Settings
→ Macro Settings
→ ☑ Enable All Macros (или Enable All Macros Except Digitally Signed)
→ [OK]
```

**После включения:** Перезагрузите файл и повторите Шаг 2 из раздела "Инициализация игры".

---

### ❌ Проблема: Кнопки не создались после SetupGame

**Решение:**

```
1. Удалите листы "Игра" и "История" если они есть:
   Правый клик на вкладке → Delete Sheet → [Delete]

2. Откройте Alt + F8 → SetupGame → [Run] еще раз

3. Если не помогло:
   - Откройте Alt + F11 (VBA редактор)
   - Проверьте что есть все 3 модуля:
     RouletteGame, GameInterface, SpreadsheetLayout
   - Если одного не хватает, добавьте его заново
   - Закройте редактор
   - Повторите SetupGame
```

---

### ❌ Проблема: "SetupGame не видна в списке макросов"

**Решение:**

```
Это означает что код не загружен правильно.

1. Откройте Alt + F11 (VBA редактор)
2. Найдите модуль SpreadsheetLayout в дереве (слева)
3. Откройте его (двойной клик)
4. Найдите в коде функцию: Sub SetupGame()
5. Если её нет - пересимпируйте копирование кода из src/SpreadsheetLayout.bas
6. Закройте редактор (Alt + Q)
7. Повторите Alt + F8 → Должен быть SetupGame
```

---

### ❌ Проблема: При клике на кнопку ничего не происходит

**Решение:**

```
1. Убедитесь что вы на листе "Игра" (нажмите вкладку)
2. Проверьте что макросы включены
3. Попробуйте:
   - Нажмите кнопку "НАЧАТЬ ИГРУ" первой
   - Если не работает, откройте VBA (Alt + F11)
   - В окне редактора найдите Sub SetupGame()
   - Нажмите на ней и нажмите F5 (Run Sub)
   - Закройте редактор
   - Вернитесь в Excel и попробуйте кнопку снова
```

---

### ❌ Проблема: "Недостаточно средств" при малой ставке

**Решение:**

```
1. Откройте лист "Игра"
2. Нажмите на ячейку B1 (начальный баланс)
3. Установите большую сумму, например 100000
4. Нажмите Enter
5. Нажмите кнопку "НАЧАТЬ ИГРУ"
6. Повторите ставку
```

---

### ❌ Проблема: Ошибка "Run-time error" при клике на кнопку

**Решение:**

```
Часто ошибка в номере ячейки. Проверьте:

1. Откройте лист "Игра"
2. Убедитесь что ячейка D3 имеет число (сумму ставки)
3. Если D3 пуста, введите число: 10 и нажмите Enter
4. Повторите клик на кнопку ставки
```

---

### ❌ Проблема: После повторного открытия файла кнопки исчезли

**Решение:**

```
Это нормально! Кнопки создаются динамически каждый раз.

1. Откройте Alt + F8
2. Запустите SetupGame еще раз
3. Кнопки появятся снова

Если хотите что бы кнопки сохранялись:
- Это требует более сложного кода (сохранение в форме вместо динамического создания)
- Свяжитесь с автором для этой функции
```
