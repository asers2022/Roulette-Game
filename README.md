# 🎰 Рулетка - VBA Игра в Excel

Полнофункциональная игра в рулетку на VBA для Microsoft Excel с визуальным интерфейсом, историей ставок и несколькими стратегиями игры.

## ✨ Возможности

✅ **Полная механика европейской рулетки** (0-36)  
✅ **10+ типов ставок** (красное/черное, четные/нечетные, дюжины, колонны, на число)  
✅ **Ставки от 1 до 100** с валидацией баланса  
✅ **Визуальный интерфейс** с кнопками на листе Excel  
✅ **История всех ставок** с цветовой разметкой (зеленый - выигрыш, красный - проигрыш)  
✅ **Статистика** (баланс, выигрыши, потери, количество спинов)  
✅ **Несколько стратегий** (Классическая, Мартингейл, Д'Аламбер)  
✅ **Автоматические вращения** (автоспин)

## 📥 Инструкция по установке

### Шаг 1: Подготовка файла Excel

1. Откройте **Microsoft Excel**
2. Создайте новую книгу (или откройте существующую)
3. Сохраните файл как **Макрос-Enabled Workbook (.xlsm)**

### Шаг 2: Добавление VBA кода

1. Нажмите **Alt + F11** чтобы открыть редактор VBA
2. В редакторе VBA создайте **новые модули** для каждого файла:
   - **RouletteGame.bas** - основная логика игры
   - **GameInterface.bas** - обработчики кнопок
   - **SpreadsheetLayout.bas** - создание интерфейса

3. Скопируйте код каждого модуля из файлов репозитория

### Шаг 3: Инициализация игры

1. Вернитесь в Excel (Ctrl + Shift + P или закройте редактор VBA)
2. Откройте **View** → **Macros** (или Alt + F8)
3. Найдите и выполните макрос **SetupGame**
4. Откроются два новых листа: "Игра" и "История" с готовым интерфейсом

### Шаг 4: Запуск игры

1. Перейдите на лист "Игра"
2. Нажмите кнопку **"НАЧАТЬ ИГРУ"**
3. Введите сумму ставки (1-100) в ячейку **D3**
4. Выбирайте тип ставки и нажимайте соответствующие кнопки

## 🎮 Как играть

### Базовые действия:

1. **Установите начальный баланс** (ячейка B1) - по умолчанию 10000
2. **Нажмите "НАЧАТЬ ИГРУ"** - инициализирует игру
3. **Введите сумму ставки** в ячейке D3 (от 1 до 100)
4. **Выберите тип ставки** и нажмите соответствующую кнопку
5. **Посмотрите результат**:
   - ✅ Если выигрыш - сумма добавляется к балансу
   - ❌ Если проигрыш - сумма вычитается из баланса
6. **История обновляется** автоматически на листе "История"

### Завершение игры:

- Нажмите **"ВЫХОД"** для закрытия игры
- Или просто закройте Excel файл

## 💰 Типы ставок

### 1. **КРАСНОЕ / ЧЕРНОЕ** (1:1)
- **Выплата**: Деньги удваиваются (ставка + выигрыш)
- **Вероятность**: 48.6% (18 из 37 чисел)
- **Пример**: Ставка 50 → Выигрыш 50 → Получите 100

### 2. **ЧЕТНОЕ / НЕЧЕТНОЕ** (1:1)
- **Выплата**: 1:1
- **Вероятность**: 48.6%
- **Пример**: Ставка 50 → Выигрыш 50

### 3. **HIGH (19-36) / LOW (1-18)** (1:1)
- **Выплата**: 1:1
- **Вероятность**: 48.6%
- **Пример**: Ставка 50 → Выигрыш 50

### 4. **ДЮЖИНЫ** (2:1)
- **ДЮЖИНА 1**: Числа 1-12
- **ДЮЖИНА 2**: Числа 13-24
- **ДЮЖИНА 3**: Числа 25-36
- **Выплата**: 2:1 (ставка × 3)
- **Вероятность**: 32.4% (12 из 37)
- **Пример**: Ставка 50 → Выигрыш 100 → Получите 150

### 5. **КОЛОННЫ** (2:1)
- **КОЛОННА 1**: 1, 4, 7, 10, 13, 16, 19, 22, 25, 28, 31, 34
- **КОЛОННА 2**: 2, 5, 8, 11, 14, 17, 20, 23, 26, 29, 32, 35
- **КОЛОННА 3**: 3, 6, 9, 12, 15, 18, 21, 24, 27, 30, 33, 36
- **Выплата**: 2:1
- **Вероятность**: 32.4%

### 6. **НА ЧИСЛО** (35:1)
- **Описание**: Ставка на конкретное число от 0 до 36
- **Выплата**: 35:1 (ставка × 36)
- **Вероятность**: 2.7% (1 из 37)
- **Пример**: Ставка 10 на число 7 → Выигрыш 350 → Получите 360

## 📊 Стратегии

### 1. **КЛАССИЧЕСКАЯ** (Рекомендуется новичкам)
- Просто ставьте одну и ту же сумму каждый раз
- Без усложняющих правил
- Низкий риск, низкий выигрыш

```
Ставка 1: 50 (Выигрыш) → Баланс +50
Ставка 2: 50 (Проигрыш) → Баланс -50
Ставка 3: 50 (Выигрыш) → Баланс +50
```

### 2. **МАРТИНГЕЙЛ** (Средний риск)
- Удвайте ставку после каждого проигрыша
- Уменьшайте до минимума после выигрыша
- **Преимущество**: Быстро восстанавливается от убытков
- **Недостаток**: Нужен большой баланс

```
Ставка 1: 50 (Проигрыш) → Баланс -50
Ставка 2: 100 (Проигрыш) → Баланс -100
Ставка 3: 200 (Выигрыш) → Баланс +200 ✓ (Восстановление)
Ставка 4: 50 (Новая серия)
```

### 3. **Д'АЛАМБЕР** (Низкий риск)
- Увеличивайте ставку на 1 после проигрыша
- Уменьшайте на 1 после выигрыша
- **Преимущество**: Более безопасно чем Мартингейл
- **Недостаток**: Медленнее восстанавливается

```
Ставка 1: 50 (Проигрыш) → Баланс -50
Ставка 2: 51 (Проигрыш) → Баланс -51
Ставка 3: 52 (Выигрыш) → Баланс +52 ✓
Ставка 4: 51 (Следующая ставка)
```

## ⚙️ Технические детали

### Используемые модули VBA:

1. **RouletteGame.bas**
   - `SpinWheel()` - генерирует случайное число (0-36)
   - `PlaceBet()` - обработка ставки и расчет выигрыша
   - `CalculateWinnings()` - расчет выплаты в зависимости от типа ставки
   - `InitializeGame()` - инициализация игры

2. **GameInterface.bas**
   - `BtnRed_Click()`, `BtnBlack_Click()` - обработчики кнопок основных ставок
   - `BtnEven_Click()`, `BtnOdd_Click()` - четные/нечетные
   - `BtnHigh_Click()`, `BtnLow_Click()` - высокие/низкие
   - `GetBetAmount()` - получение суммы ставки из ячейки

3. **SpreadsheetLayout.bas**
   - `CreateGameLayout()` - создает оба листа и устанавливает дизайн
   - `SetupGameSheet()` - настраивает лист "Игра"
   - `SetupHistorySheet()` - настраивает лист "История"
   - `CreateButton()` - создает кнопки с макросами
