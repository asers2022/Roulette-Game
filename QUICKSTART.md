# ⚡ Быстрый старт - Рулетка VBA

Инструкция за 5 минут для запуска игры.

---

## 🚀 Быстрая установка (5 минут)

### Шаг 1️⃣: Загрузите файлы кода

1. Откройте репозиторий: https://github.com/asers2022/Roulette-Game
2. Нажмите **Code** → **Download ZIP**
3. Распакуйте архив
4. Откройте файлы из папки `src/`:
   - `RouletteGame.bas`
   - `GameInterface.bas`
   - `SpreadsheetLayout.bas`

---

### Шаг 2️⃣: Создайте файл Excel

1. Откройте **Microsoft Excel**
2. Создайте новую книгу
3. **Сохраните как**: `Roulette_Game.xlsm` (Важно! .xlsm формат)
   - File → Save As
   - Формат: **Excel Macro-Enabled Workbook (.xlsm)**

---

### Шаг 3️⃣: Добавьте VBA код

1. Нажмите **Alt + F11** (откроется редактор VBA)
2. В редакторе VBA: **Insert** → **Module**
3. Создайте 3 новых модуля и скопируйте в каждый:
   - **Module1**: Содержимое `RouletteGame.bas`
   - **Module2**: Содержимое `GameInterface.bas`
   - **Module3**: Содержимое `SpreadsheetLayout.bas`

**Как скопировать:**
- Откройте файл .bas в текстовом редакторе
- Ctrl+A → Ctrl+C (скопировать весь код)
- В VBA редакторе → Ctrl+V (вставить)

---

### Шаг 4️⃣: Запустите инициализацию

1. Закройте редактор VBA (Alt+Q или крестик)
2. Вернулись в Excel
3. Нажмите **Alt + F8** (Macros dialog)
4. Найдите макрос **SetupGame**
5. Нажмите **Run** (зеленая кнопка)

**Результат:** 
- ✅ Создадутся два новых листа: "Игра" и "История"
- ✅ Появятся все кнопки и элементы управления

---

### Шаг 5️⃣: Начните играть! 🎰

1. На листе **"Игра"** нажмите кнопку **"НАЧАТЬ ИГРУ"**
2. Введите сумму ставки в ячейке **D3** (от 1 до 100)
3. Выберите тип ставки:
   - 🔴 **КРАСНОЕ** / ⚫ **ЧЕРНОЕ**
   - 🟣 **ЧЕТНОЕ** / 🟠 **НЕЧЕТНОЕ**
   - Или любую другую ставку
4. Смотрите результат!
5. История ставок появляется на листе **"История"**

```
ЛЕВАЯ ЧАСТЬ (A:B)        СЕРЕДИНА (D:E)        ПРАВАЯ ЧАСТЬ (F:M)

```

## ⚙️ Если что-то не работает

### Проблема: Кнопки не создались

**Решение:**
1. Проверьте, сохранен ли файл как **.xlsm**
2. Включите макросы: File → Options → Trust Center → Trust Center Settings → Macro Settings → Enable All Macros
3. Запустите **SetupGame** снова

### Проблема: "Макрос не найден"

**Решение:**
1. Убедитесь, что вы скопировали ВСЕ три модуля
2. Проверьте, что нет опечаток в именах модулей
3. Перезагрузите Excel

### Проблема: При клике на кнопку ничего не происходит

**Решение:**
1. Проверьте, что макросы включены
2. Попробуйте Ctrl+Alt+Delete → Task Manager → перезагрузите Excel
3. Убедитесь, что вы находитесь на листе "Игра"

### Проблема: "Недостаточно средств"

**Решение:**
1. Увеличьте начальный баланс в ячейке B1
2. Нажмите "НАЧАТЬ ИГРУ" еще раз
3. Или уменьшите сумму ставки в ячейке D3

## 🎯 Стратегии для опытных игроков

### Мартингейл (Risky ⚠️)
- Удваивайте ставку после проигрыша
- Пример: 10 → 20 → 40 → 80
- **Выберите**: Кнопка "МАРТИНГЕЙЛ"

### Д'Аламбер (Safe ✅)
- Увеличивайте на 1 после проигрыша
- Пример: 10 → 11 → 12 → 13
- **Выберите**: Кнопка "Д'АЛАМБЕР"

### Классическая (Balanced 💎)
- Одна и та же ставка
- Пример: 10 → 10 → 10 → 10
- **Выберите**: Кнопка "КЛАССИЧЕСКАЯ"
